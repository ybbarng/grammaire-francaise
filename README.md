# Grammaire Française

Claude Code와 대화형으로 프랑스어 문법을 학습하는 프로젝트입니다.

## 목표
1. A1부터 C2(문어체 포함)까지 프랑스어 문법을 체계적으로 완주
2. 프랑스어 문장을 자유롭게 구사할 수 있는 수준 도달

## 학습 방식
- 정형화된 퀴즈가 아닌 **대화형 수업 + 피드백**
- 슬래시 커맨드로 세션 관리 (`/project:study`, `/project:review`, `/project:write`, `/project:end`)
- 세션마다 진도와 취약점을 자동 기록

## 커리큘럼 (15장 49항목)

| 단계 | 범위 | 핵심 |
|------|------|------|
| 초급 (A1-A2) | 1~5장 | 문장을 "정의하고 생성" |
| 중급 (B1-B2) | 6~10장 | 문장을 "연결·변형" |
| 고급 (C1+) | 11~15장 | 문장을 "설계·압축·조정" |

## 구조

```
├── progress.md          # 커리큘럼 진도표
├── sessions/            # 세션별 상세 기록
├── exercises/           # 작문 연습 및 교정 기록
├── docs/notion-export/  # 기존 문법 자료 (Notion export)
└── .claude/commands/    # 슬래시 커맨드 정의
```
