프랑스어 문법 학습 세션을 시작한다.

## 절차 (반드시 순서대로 수행)

### 1. 진도 확인
- `progress.md`를 읽고 현재 상태를 파악한다.
- 마지막 세션 기록을 확인한다 (`sessions/` 디렉토리의 가장 최근 파일).
- 이전 세션의 취약점이나 복습 필요 항목이 있으면 먼저 언급한다.

### 2. 주제 선택
- `progress.md`의 커리큘럼에서 다음 학습 주제를 제안한다.
- 사용자가 다른 주제를 원하면 그것을 따른다.

### 3. 학습 자료 참고
- 해당 주제의 보강 자료(`docs/materials/`)를 우선 읽는다.
- 보강 자료가 없으면 Notion 원본(`docs/notion-export/`)을 참고한다.
- 자료를 기반으로, 부족한 부분을 보충하며 설명한다.

### 4. 수업 진행
- 개념 설명 → 예문 제시 → 사용자 연습 → 피드백 순서로 진행한다.
- 언어 전환 전략을 따른다:
  - A1-A2 주제: 한국어 설명, 예문만 프랑스어
  - B1-B2 주제: 간단한 프랑스어 + 한국어 보조
  - C1+ 주제: 프랑스어 위주, 모르는 부분만 한국어
- 사용자가 프랑스어로 시도하면 적극적으로 프랑스어로 응답한다.

### 5. 세션 중 기록
- 사용자가 틀린 내용, 교정 사항, 반복 필요한 패턴을 메모해둔다.
- 세션이 끝나면 `/project:end`를 안내한다.
