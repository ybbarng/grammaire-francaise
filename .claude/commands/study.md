프랑스어 문법 학습 세션을 시작한다.

## 절차 (반드시 순서대로 수행)

### 1. 진도 확인 및 페이스 점검
- `progress.md`를 읽고 현재 상태를 파악한다.
- 마지막 세션 기록을 확인한다 (`sessions/` 디렉토리의 가장 최근 파일).
- **페이스 점검**: `README.md`의 계획표와 현재 진도를 비교하여 사용자에게 보고한다:
  - 현재 날짜 기준으로 계획 대비 진행 상황 (앞서감/정상/뒤처짐)
  - 뒤처진 경우: 따라잡기 위해 필요한 페이스 제안
  - 앞서가는 경우: 여유 시간 활용 방안 제안 (복습/작문 등)

### 2. 시험 확인
- 오늘 날짜 기준으로 예정된 시험이 있는지 `progress.md`의 마일스톤을 확인한다.
- 시험이 예정된 경우, 해당 시험을 우선 실시한다:
  - **묶음 소시험**: 해당 묶음의 마지막 주제 학습이 끝난 직후 실시.
    - 10문제, 통과 기준 100%. 이월 오답이 있으면 문제 수 추가.
    - 미통과 시: 틀린 부분을 취약점에 기록하고, 다음 소시험에 이월.
  - **단계별 평가**: 초급/중급/고급 완료 시 실시.
    - 20문제, 통과 기준 95%.
    - 미통과 시: 틀린 부분 복습 후 다음 세션에서 재시험.
  - **월말 누적 평가**: 매월 25일에 1차 실시.
    - 25문제, 통과 기준 95%.
    - 1차 미통과 시: 틀린 부분 복습, 말일에 2차 시험.
    - 2차도 미통과 시: 취약점 기록 후 다음 달 학습에서 보완.
- 시험 결과는 `progress.md` 마일스톤 상태에 기록한다.
- 시험이 없으면 다음 단계로 넘어간다.

### 3. 간격 반복 워밍업
- `progress.md`의 "복습 스케줄" 섹션을 확인한다.
- 오늘 날짜 기준으로 복습 예정일이 된 주제가 있으면:
  - 해당 주제에서 2~3문제를 프랑스어로 출제한다 (이전 세션에서 틀렸던 유형 우선).
  - 결과에 따라 이해도와 다음 복습 예정일을 조정한다.
- 복습 대상이 없으면 바로 새 주제로 넘어간다.

### 4. 주제 선택
- `progress.md`의 커리큘럼에서 다음 학습 주제를 제안한다.
- 사용자가 다른 주제를 원하면 그것을 따른다.

### 5. 학습 자료 참고
- 해당 주제의 보강 자료(`docs/materials/`)를 우선 읽는다.
- 보강 자료가 없으면 Notion 원본(`docs/notion-export/`)을 참고한다.
- 자료를 기반으로, 부족한 부분을 보충하며 설명한다.

### 6. 수업 진행
- **전체 프랑스어로 진행한다.** 프랑스 현지 원어민 선생님처럼 대화한다.
- 개념 설명 → 예문 제시 → 사용자 연습 → 피드백 순서로 진행한다.
- 한 개념씩 작은 단위로 제시하고, 사용자가 질문하거나 다음을 요청하면 진행한다.
- 사용자의 프랑스어 문장에 오류가 있으면:
  1. 먼저 프랑스어로 교정 및 설명
  2. 사용자가 이해 못하면 한국어로 보충
- 사용자가 한국어로 질문하면 한국어로 답해도 되지만, 가능하면 프랑스어로 돌아온다.
- 연습/퀴즈는 **가르친 범위 내에서만** 출제한다 (가르치지 않은 어휘/패턴 요구 금지).
- 새 주제 연습 시 이전 주제의 구조를 섞은 누적 연습을 포함한다.

### 7. 소시험 실시 (해당 시)
- 해당 주제가 묶음의 마지막 주제라면, 연습 후 소시험을 실시한다.
- 소시험 절차는 위 2번 참고.

### 8. 세션 중 기록
- 사용자가 틀린 내용, 교정 사항, 반복 필요한 패턴을 메모해둔다.
- 세션이 끝나면 `/project:end`를 안내한다.
