# Français Grammaire — 대화형 프랑스어 문법 학습

## 프로젝트 목적
1. 프랑스어 문법을 A1부터 C2(문어체 포함)까지 완벽히 체계적으로 학습
2. 프랑스어 문장을 자유롭게 구사할 수 있는 수준 도달

## 학습 방식
- **웹앱 아님** — Claude와 대화형으로 수업 + 피드백
- 정형화된 퀴즈보다 **설명 → 연습 → 교정** 흐름
- `docs/notion-export/` 에 기존 ChatGPT와 정리한 문법 자료 있음 (Notion export)

## 언어 전환 전략
- Phase 1 (A1-A2 복습): 한국어로 설명, 예문만 프랑스어
- Phase 2 (B1-B2): 간단한 프랑스어 + 한국어 보조
- Phase 3 (C1+): 프랑스어 위주, 모르는 부분만 한국어
- 사용자가 프랑스어로 시도하면 적극적으로 프랑스어로 응답

## 세션 진행 규칙
1. 세션 시작 시 `progress.md`를 읽고 이전 진도 확인
2. 해당 주제의 Notion 자료(`docs/notion-export/`)를 먼저 참고
3. 수업: 개념 설명 → 예문 → 사용자 연습 → 피드백
4. 세션 종료 시 `progress.md` 업데이트 (진도, 취약점, 다음 주제)

## 커리큘럼 구조 (Notion 기반)

### 초급 (A1-A2) — 문장을 "정의하고 생성"
- 1장: 문장 공리 & 기본 구조
- 2장: 명사 시스템 (Nominal System)
- 3장: 형용사 시스템
- 4장: 동사 기초 구조
- 5장: 기본 전치사 & 부사

### 중급 (B1-B2) — 문장을 "연결·변형"
- 6장: 목적어 구조 & 대명사
- 7장: 시제 논리
- 8장: 관계대명사 기초
- 9장: 접속법 기초
- 10장: 문장 변형

### 고급 (C1+) — 문장을 "설계·압축·조정"
- 11장: 관계대명사 심화
- 12장: 분사 & 축약
- 13장: 고급 접속법 & 법
- 14장: 추상 대명사 & 담화
- 15장: 문어체 전용 문법

## 파일 구조
- `CLAUDE.md` — 이 파일. 프로젝트 컨텍스트
- `progress.md` — 학습 진도 추적
- `docs/notion-export/` — Notion에서 가져온 기존 문법 자료
- `exercises/` — 작문 연습 및 교정 기록
