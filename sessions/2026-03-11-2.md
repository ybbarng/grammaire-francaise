## 새로 배운 개념
1. **부정은 동사(사건)를 부정한다**: 주어나 목적어가 아니라, 동사에 작용하는 연산
2. **ne...pas 구조**: ne는 동사 앞, pas는 동사 뒤 — 동사를 감싸는 형태
3. **ne와 pas의 역할 분리**: ne = 문법적 위치 표식 (구어에서 탈락 가능), pas = 의미적 부정 핵심
4. **pas의 어원**: 라틴어 passum(걸음)에서 유래 → "한 걸음도 ~않다" → 일반적 부정어로 확대 (Jespersen 순환)
5. **복합 시제 부정**: 활용된 동사(조동사)를 감쌈 — *Je n'ai pas mangé.*
6. **대명사 + 부정**: ne는 대명사 앞, pas는 동사 뒤 — *Je ne me lève pas.*
7. **다른 부정어**: jamais/plus/rien/personne가 pas를 **대체** (함께 쓰지 않음)
8. **부정 + 도치**: Ne + [동사-주어대명사] + pas — 동사와 주어대명사 블록은 분리 불가
9. **대명사 + 부정 + 도치**: Ne + (목적어 대명사들) + [동사-주어대명사] + pas

## 틀린 것과 교정
- **부정어 대체 원리 혼동**: jamais/rien 등이 souvent/quelque chose를 **대체**하는 것이지, 함께 쓰는 것이 아님. 연습 중 2회 오류 후 이해.
- **élision 여전히 취약**: 소시험에서 *est-ce que il* → *est-ce qu'il* 오류 미발견 (이전 세션에서도 같은 실수)

## 프랑스어 표현 교정
- *Touts* → **Tout**, *touts chaque éléments* → **tous les éléments** / **chaque élément**
- *à le sujet* → **au sujet** (à + le = au)
- *diréction* → **direction**
- *le signification* → **la signification** (여성 명사)
- *a devenu* → **est devenu** (devenir는 être 조동사 사용)
- *expension* → **expansion**
- *Qu'est ce que on parle* → **Comment est-ce qu'on dit** (comment/que 구별, élision)
- *parler une phrase* → **dire une phrase**
- *Comment pour les multiple pronoms* → **Comment fait-on avec plusieurs pronoms**
- *les plusieurs objectives pronoms* → **plusieurs pronoms objets**
- *J'ai une question la forme pour* → **J'ai une question sur la forme de**
- *entorer* → **entourer**
- *le passé conposé* → **le participe passé** (시제명 vs 과거분사 구별)
- *Comment je peux savoir* → **Comment est-ce que je peux savoir**
- *doit joindre tout les deux conditiions* → **doit combiner les deux conditions**
- *la neative* → **la négation**

## 반복 필요 항목
- élision (est-ce que + 모음 → est-ce qu'): 소시험에서도 여전히 실수
- 부정어가 기존 단어를 대체하는 원리 (jamais ↔ souvent, rien ↔ quelque chose)
- à + le → au 축약
