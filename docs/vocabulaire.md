# Vocabulaire — 학습 어휘 풀

시험 출제 시 이 목록에 있는 어휘만 사용한다. 새 어휘가 필요하면 문제에 뜻을 병기한다.
세션에서 예문으로 등장한 어휘를 누적 기록한다.

---

## 1장: 문장 공리 & 기본 구조

### 1-1. 프랑스어 문장의 최소 조건

| 단어 | 품사 | 성 | 뜻 |
|------|------|----|----|
| je | 대명사 | - | 나 |
| moi | 대명사 (강세형) | - | 나 |
| il | 대명사 | 남 | 그 / 비인칭 주어 |
| suis | 동사 (être 1인칭 단수) | - | ~이다, ~있다 |
| fait | 동사 (faire 3인칭 단수) | - | 하다, 만들다 |
| pleut | 동사 (pleuvoir 3인칭 단수) | - | 비가 오다 |
| faut | 동사 (falloir 3인칭 단수) | - | ~해야 한다 |
| mangez | 동사 (manger 2인칭 복수/명령형) | - | 먹다 |
| fatigué | 형용사 | 남 | 피곤한 |
| froid | 형용사 | 남 | 추운, 차가운 |
| chaud | 형용사 | 남 | 더운, 뜨거운 |
| beau | 형용사 | 남 | 아름다운, (날씨가) 좋은 |
| étudiant | 명사 | 남 | 학생 |
| à | 전치사 | - | ~에, ~에서 |
| aujourd'hui | 부사 | - | 오늘 |
| il y a | 표현 | - | ~가 있다 |
| il fait + 형용사 | 표현 | - | (날씨가) ~하다 |
