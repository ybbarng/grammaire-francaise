# 5️⃣-3. 부사의 형성

### (Formation des adverbes)

> 부사는 왜 대부분 -ment로 끝나는가
> 
> 
> 왜 여성형이 기준이 되는가
> 

---

## 📌 핵심 명제 (정의)

> 프랑스어 부사는
형용사의 여성형을 기반으로
-ment를 붙여 형성된다.
> 

이건 관습이 아니라 **형태적 논리**다.

---

## 1️⃣ 기본 규칙 (90% 이상)

### 🔹 형용사 → 여성형 → + ment

공식:

```
형용사 (féminin) + ment

```

### 예문

- lent → lente → **lentement**
- heureux → heureuse → **heureusement**
- sérieux → sérieuse → **sérieusement**

👉 **여성형이 기준**이라는 점이 핵심

---

## 2️⃣ 왜 하필 여성형일까? (이유 설명)

이건 아주 중요한 질문이고, 답은 단순해.

> 여성형은
형용사의 ‘기본 어간’을 가장 잘 보존하는 형태다.
> 

### 비교해보자

- heureux
    - 남성형: heureux (발음 약화)
    - 여성형: heureuse (형태 명확)

👉 여성형이:

- 자음 보존
- 형태 안정

그래서:

> 부사 = 형용사 어간을 최대한 유지한 형태
> 

---

## 3️⃣ -ant / -ent 형용사 (중요 예외군)

### 규칙

| 형용사 | 부사 |
| --- | --- |
| constant | constamment |
| récent | récemment |

⚠️ 주의:

- **amment**
- **emment**

👉 발음은 둘 다 **[amɑ̃]**

---

## 4️⃣ -e로 끝나는 형용사

이 경우는 더 단순하다.

> 형용사 + ment
> 
- rapide → **rapidement**
- facile → **facilement**
- énorme → **énormément**

👉 이미 여성형 = 남성형

---

## 5️⃣ 불규칙 부사 (암기 대상)

이건 이유 설명보다 **빈도 문제**다.

| 형용사 | 부사 |
| --- | --- |
| bon | **bien** |
| mauvais | **mal** |
| meilleur | **mieux** |
| pire | **pis** (제한적) |

👉 아주 자주 쓰임

👉 **암기 필요**

---

## 6️⃣ 형용사 = 부사인 경우 (중요)

일부 형용사는 **그대로 부사로 사용**된다.

- parler **vite**
- coûter **cher**
- travailler **dur**

👉 성·수 변화 ❌

👉 부사로 쓰일 때는 **불변**

---

## 7️⃣ 지금 단계에서의 학습 포인트

### ✔ 지금 해야 할 것

- 기본 형성 규칙 이해
- 왜 여성형인지 납득
- 불규칙 핵심 몇 개 암기

### ❌ 지금 안 해도 되는 것

- 희귀 부사
- 문어체 부사
- 의미 미묘한 차이

---

## 🧠 5-3 요약 (이걸 말할 수 있으면 통과)

- 부사는 **형용사 여성형 + ment**
- 여성형은 **형태 안정성 때문**
- ant / -ent → -amment / -emment
- 일부는 불규칙
- 일부는 형용사 그대로 사용

---

## 🎓 5장 전체 완료 체크

이제 이 질문들에 답할 수 있으면 **5장 끝**이야:

1. 왜 부사는 성·수 일치를 안 하는가?
2. 왜 대부분의 부사는 -ment로 끝나는가?
3. 왜 여성형이 기준이 되는가?
