# 9-4. subjonctif présent의 굴절 구조

# 9️⃣-4. subjonctif présent의 굴절 구조

*(의미가 만든 형태)*

> 접속법 활용은
> 
> 
> 외워야 할 표가 아니라,
> 
> **이미 결정된 mood를
> 동사에 실현하는 방식**이다
> 

---

## 📌 핵심 명제 (정의 · 공리)

> subjonctif présent의 굴절은
새로운 시제 체계가 아니라,
이미 선택된 ‘비사실 모드’를
동사 형태로 구현한 결과다.
> 
- 시간 표현 ❌
- 사건 순서 ❌
- 핵심은:
    
    > mood의 형태적 실현
    > 

👉 그래서 **의미·구조가 먼저**,

👉 **굴절은 나중**이다.

---

## 1️⃣ 굴절을 보기 전에 반드시 고정할 것

이 질문에 즉답할 수 있어야 한다.

> “이 문장은 왜 접속법인가?”
> 
- YES → 굴절 학습 가능
- NO → 아직 이르다

👉 굴절은 **판단 도구가 아니라 출력 형식**이다.

---

## 2️⃣ subjonctif présent의 기본 생성 원리

### 🔹 핵심 규칙 (중요)

> 접속법 현재는
직설법 현재 3인칭 복수형의 어간을 사용한다.
> 

이 한 줄이 **전체의 80%**다.

---

### 🔹 예시

| 동사 | 직설법 3인칭 복수 | 접속법 어간 |
| --- | --- | --- |
| parler | parlent | parl- |
| finir | finissent | finiss- |
| prendre | prennent | prenn- |
| venir | viennent | vienn- |

👉 **귀로 들리는 어간**이 기준

---

## 3️⃣ 접속법 현재 어미

어미는 **규칙적**이다.

```
-e   -es   -e   -ions   -iez   -ent

```

| 인칭 | 어미 |
| --- | --- |
| je | -e |
| tu | -es |
| il/elle/on | -e |
| nous | -ions |
| vous | -iez |
| ils/elles | -ent |

👉 **불규칙은 어간**,

👉 **어미는 규칙**이다.

---

## 4️⃣ 예외처럼 보이는 핵심 불규칙 동사들

### 🔹 être

| 인칭 | 형태 |
| --- | --- |
| je | sois |
| tu | sois |
| il | soit |
| nous | soyons |
| vous | soyez |
| ils | soient |

---

### 🔹 avoir

| 인칭 | 형태 |
| --- | --- |
| j’ | aie |
| tu | aies |
| il | ait |
| nous | ayons |
| vous | ayez |
| ils | aient |

---

### 🔹 aller / faire / pouvoir (요약)

- aller → *aille / allions*
- faire → *fasse / fassions*
- pouvoir → *puisse / puissions*

👉 **빈도 높은 동사만 별도 암기**

---

## 5️⃣ 왜 nous / vous 어미가 다를까?

여기서 형태의 논리가 드러난다.

### 🔹 관찰

- je / tu / il / ils → **짧고 구어적**
- nous / vous → **명확한 어미**

이유:

> 접속법은
구어에서 ‘구분이 들리지 않기 때문에’
nous / vous에서만 형태 차이를 유지했다.
> 

👉 기능적 타협

---

## 6️⃣ 직설법과 형태가 같은 경우 (중요)

많은 동사는:

- **직설법 현재 = 접속법 현재** (발음 기준)

예:

- *qu’il parle*
- *qu’ils parlent*

👉 차이는 **형태가 아니라 구조**

그래서:

> ❌ “형태로 접속법을 판별”
> 
> 
> ⭕ **구조로 이미 판별**
> 

---

## 7️⃣ 접속법 활용을 외우는 올바른 전략

### 🔹 잘못된 전략

- ❌ 동사별 접속법 표 암기
- ❌ 직설법과 비교 암기

---

### 🔹 올바른 전략

1. **구조로 접속법 필요성 판단**
2. 3인칭 복수형 어간 추출
3. 규칙 어미 적용
4. 핵심 불규칙만 암기

👉 이 순서가 **인지 부하 최소**

---

## 8️⃣ 왜 접속법 굴절은 ‘적게’ 남았는가?

역사적 이유를 한 줄로 정리하면 이거다.

> 접속법은
시제가 아니라 mood이기 때문에
형태 확장이 최소화되었다.
> 

그래서:

- 현재 → 있음
- 과거 → 제한적
- 미래 ❌

👉 **기능에 비해 형태는 최소**

---

## 🧠 9-4 요약

- 접속법 굴절 = **의미의 결과**
- 어간 = 직설법 3인칭 복수
- 어미 = 규칙적
- 불규칙은 **빈도 기반**
- 형태로 판별 ❌ / 구조로 판별 ⭕

---

## 🎓 9-4 완료 기준

다음을 할 수 있으면 **9-4 완료**:

- 접속법이 필요한 문장을 **활용표 없이** 판단한다
- 3인칭 복수형에서 **어간을 즉시 뽑는다**
- être / avoir / faire를 **막힘없이 활용한다**
- “왜 이 형태가 나왔는지” 설명할 수 있다

---

이제 접속법의 **기본 체계는 완주**다.
